Libri Vermis

Course: Orientation Project - 1st year (2013-2014)
Tools: HTML, CSS, PHP, SQL, Photoshop
Type: Group work
Link: Libri Vermis
The aim of the project was to create a social website to motivate people to read books. In order to use the website users are required to create an account. Once the registration is successful, logged in users can create and edit a book wishlist and a "bookshelf" list, which includes the "reading", "read", "on hold" and "dropped" subsections. Books can be added to the lists by using the "search" function of the website, linked to the database of the project. If not present, the book information can be added manually. Moreover users can add friends, exchange messages and share their lists.

RealBirthDay

Course: Mobile Programming - 2nd year (2014-2015)
Tools: HTML, CSS, Bootstrap, Javascript, jQuery
Type: Pair work
Link: RealBirthDay
The aim of the project was to practice the use of APIs, in particular the one from Facebook. Users can log in to the app through the Facebook Login. Once the required authorizations have been granted, users can view a list of their friends which includes their profile picture, their birthday and a form allowing the user to add and edit notes for each friend.

Rhino (temporary name)

Course: Innovation Project - 2nd year (2014-2015)
Tools: HTML, CSS, Bootstrap, Javascript, jQuery, PHP, SQL
Type: Independent work as part of bigger project
Link: WIP
The project is carried out as a part of Arend Project, which is aimed at wildlife conservation and involves four international teams from the United States, South Africa, Germany, and Finland. The task assigned to the Metropolia team consists in the development of a ground sensor network and of a web-based system to gather, display and store data sent by the network.
My role is focused on the creation of a web system able to exchange data with the ground network and elaborate to a human-readable form the received information, which is then displayed to the user and stored to a database. The design and creation of the database are also a part of my assigned tasks.
